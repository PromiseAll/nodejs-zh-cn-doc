稳定性: 3 - 旧版

源代码: lib/querystring.js

querystring 模块提供了用于解析和格式化网址查询字符串的实用工具。 可以使用以下方式访问它：

const querystring = require('querystring');

querystring API 被视为旧版的。 虽然它仍在维护，但是新的代码应该改为使用 <URLSearchParams> API。

Stability: 3 - Legacy

Source Code: lib/querystring.js

The querystring module provides utilities for parsing and formatting URL query strings. It can be accessed using:

const querystring = require('querystring');

The querystring API is considered Legacy. While it is still maintained, new code should use the <URLSearchParams> API instead.
