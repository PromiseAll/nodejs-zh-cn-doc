v15.0.0

Added in: v15.0.0

The timers/promises API provides an alternative set of timer functions that return Promise objects. The API is accessible via require('timers/promises').

import {
  setTimeout,
  setImmediate,
  setInterval,
} from 'timers/promises';const {
  setTimeout,
  setImmediate,
  setInterval,
} = require('timers/promises');
