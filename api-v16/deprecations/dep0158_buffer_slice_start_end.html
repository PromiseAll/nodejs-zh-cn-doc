版本变更
v16.15.0

仅文档弃用。

类型: 仅文档

此方法已被弃用，因为它与 Uint8Array.prototype.slice() 不兼容，Uint8Array.prototype.slice() 是 Buffer 的超类。

使用 buffer.subarray 代替它做同样的事情。

History
VersionChanges
v16.15.0

Documentation-only deprecation.

Type: Documentation-only

This method was deprecated because it is not compatible with Uint8Array.prototype.slice(), which is a superclass of Buffer.

Use buffer.subarray which does the same thing instead.
