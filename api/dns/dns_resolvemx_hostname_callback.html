v18.0.0

v0.1.27

新增于: v0.1.27

  • hostname <string>
  • callback <Function>
    • err <Error>
    • addresses <Object[]>

使用域名系统协议解析 hostname 的邮件交换记录（MX 记录）。 传给 callback 函数的 addresses 参数将包含其中包含 priority 和 exchange 属性（例如 [{priority: 10, exchange: 'mx.example.com'}, ...]）的对象数组。

History
VersionChanges
v18.0.0

Passing an invalid callback to the callback argument now throws ERR_INVALID_ARG_TYPE instead of ERR_INVALID_CALLBACK.

v0.1.27

Added in: v0.1.27

  • hostname <string>
  • callback <Function>
    • err <Error>
    • addresses <Object[]>

Uses the DNS protocol to resolve mail exchange records (MX records) for the hostname. The addresses argument passed to the callback function will contain an array of objects containing both a priority and exchange property (e.g. [{priority: 10, exchange: 'mx.example.com'}, ...]).
